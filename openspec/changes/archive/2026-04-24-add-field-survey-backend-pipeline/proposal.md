# Change: Add FieldSurvey Backend Pipeline & Spatial Integration

## Why
This is a wildly ambitious and cutting-edge architecture. Fusing logical network topology (SNMP/WLC/Flows) with physical cyber-spatial reality (LiDAR/RF RSSI/ARKit) using high-performance IPC (Arrow/Rust) is the holy grail of network observability. The ServiceRadar backend must handle the brutal volume of data generated by the FieldSurvey iOS app while keeping the God-View UI buttery smooth.

## What Changes
1. **PostGIS Integration**: Add native support for 3D geometries (`geometry(PointZ, SRID)`) to execute 3D bounding box intersections (`&&&` operator) and distance calculations (`ST_3DDistance`).
2. **Object Storage for Meshes**: Prevent database bloat by storing the raw LiDAR meshes/USDZ files directly to an S3-compatible Object Store (MinIO/AWS S3). The DB will only store a reference/URL.
3. **TimescaleDB for High-Velocity Telemetry**: Shred the incoming Apache Arrow RecordBatches and bulk-insert the raw RF telemetry into a Timescale Hypertable for historical playback and analytics.
4. **Rust Multilateration Engine**: The Rust backend will asynchronously query the Hypertable, group by BSSID, and use the Log-Distance Path Loss model against varying (X,Y,Z) positions to triangulate the actual physical location of the Access Point.
5. **Apache AGE Graph Fusion**: Once Rust calculates the AP's physical location, it updates the Apache AGE graph to map the logical node (`AccessPoint`) to the physical node (`PhysicalLocation`).
6. **Deck.gl Frontend Mapping**: The web UI fetches the room's LiDAR Point Cloud directly from Object Storage (`PointCloudLayer`) and fetches the aggregated PostGIS AP locations and signal coverage as Apache Arrow tables to render glowing spheres (`HexagonLayer` or `ScatterplotLayer`).
7. **Future Scope (pgvector)**: Investigate embedding vectorization for RF Fingerprinting, treating an RF snapshot as an embedding to instantly calculate a user's real-time physical location indoors via K-Nearest Neighbors (KNN).

## Impact
- Affected code:
  - Database provisioning (`PostGIS`, `TimescaleDB`, `pgvector`).
  - Elixir pipeline and S3 bucket configurations.
  - Rust NIF spatial join implementations.
  - `web-ng` WebGPU Deck.gl layers.
